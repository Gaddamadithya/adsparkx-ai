# Account Security & Multi-Factor Authentication Settings

Learn how to configure your account settings, secure your login details, and set up Multi-Factor Authentication (MFA).

## 1. Setting Up Two-Factor Authentication (2FA)

We strongly recommend enabling 2FA for all accounts. We support Time-based One-Time Passwords (TOTP) via authenticator apps like Google Authenticator, Authy, or 1Password.

### Enable 2FA:
1. Log in to the Adsparkx Dashboard.
2. Go to **Settings** -> **Security** -> **Two-Factor Authentication**.
3. Scan the QR code using your authenticator application.
4. Input the 6-digit verification code generated by your app.
5. Save your backup codes in a safe place. If you lose access to your device, you will need these codes to log in.

---

## 2. API Key Rotation

Your API keys grant access to your account resources. Treat them with the same confidentiality as your password.
*   **Rotation Schedule:** We recommend rotating your API keys every **90 days**.
*   **How to Rotate:** Create a new key in the Developer Settings tab, update your services to use this new key, test that integration is working, and then delete the old key.
*   **Leaked Keys:** If you accidentally commit a key to GitHub, our system will automatically detect the leak, deactivate the key immediately, and send you a critical email notification.
